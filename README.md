# 🚗 Endless Driver

An endless runner game where you control a car and try to survive as long as possible while avoiding obstacles.

## 🎮 Gameplay

- The game is infinite — there is no finish line.
- Your goal is to avoid obstacles and achieve the highest score.
- If you crash into an obstacle, the game is over.

## 🕹 Controls

- **A** — Move left  
- **D** — Move right  
- **Space** — Jump  
- **ESC** — Open pause menu  

## 📋 Features

- Main menu with:
  - Start Game
  - Exit
- Endless gameplay
- Score system
- Game over on collision
- Pause menu (ESC):
  - Replay
  - Quit

## 🛠 Built With

- Unity (C#)

## 🚀 How to Run

1. Open the project in Unity Hub  
2. Select the project folder  
3. Click **Play** in the Unity Editor
